# Office Scheduling System

This is a simple office scheduling system that allows team members to submit their availability and view the weekly schedule.

## Setup Instructions

1. **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/office-scheduling.git
    ```

2. **Navigate to the project directory:**
    ```bash
    cd office-scheduling
    ```

3. **Install the dependencies:**
    ```bash
    npm install
    ```

4. **Start the server:**
    ```bash
    npm start
    ```

5. **Open your browser and navigate to:**
    ```
    http://localhost:3000
    ```

## How to Use

- **Submit Availability:** Fill out the form with your name and available days, then click 'Submit'.
- **View Schedule:** The weekly schedule will be displayed below the form.
